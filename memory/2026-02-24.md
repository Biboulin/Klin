## Architecture Decisions

- **Backend:** Express.js (simple, proven, easy to deploy to Vercel)
- **ORM:** Prisma (type-safe, migrations, developer experience)
- **Database:** PostgreSQL (Supabase for managed, free tier)
- **Frontend:** Expo (cross-platform, faster than native)
- **Photos:** Cloudinary URLs in DB
- **Notifications:** Firebase Cloud Messaging (Phase 2)
- **Freemium:** Enforced at API level with group.isPremium check
